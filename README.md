# Digital Store - Back-End API

Projeto Final Geração Tech. A API gerencia o controle de produtos e o cadastro de usuários.

## Tecnologias Utilizadas
* **Node.js** & **Express** (Servidor)
* **MySQL** (Banco de Dados)
* **Sequelize** (ORM para persistência de dados)
* **CORS** (Segurança e acesso)
* **Dotenv** (Gerenciamento de variáveis de ambiente)

## Como rodar o projeto
1. Clone este repositório.
2. Certifique-se de que o **XAMPP** (MySQL) está rodando.
3. Crie um banco de dados chamado `digital_store`.
4. Instale as dependências:
